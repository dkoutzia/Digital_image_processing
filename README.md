# Digital Image Processing Projects

This repository contains three projects developed for the Digital Image Processing course.

## Projects

### Project 1 – RAW to RGB Conversion
Implementation of a complete RAW image processing pipeline for converting `.DNG` Bayer-pattern images into RGB images using MATLAB.

### Project 2 – OCR using Fourier Descriptors
Implementation of an Optical Character Recognition (OCR) pipeline using contour extraction, Fourier descriptors, and character segmentation techniques.

### Project 3 – Image Registration and Feature Matching
Implementation of feature detection, local descriptors, and image registration techniques using Harris corner detection and rotation-invariant descriptors.

Each project contains its own detailed README with additional information and implementation details.
